01 / 核心

先记住一句话
布局就是把块排成你要的样子

核心

这一课只看排列本身：横着排、竖着排、排成网格、让某一块占满整行。你先会“怎么摆”，再去记“用什么属性摆”。

02 / 一行排列

先把元素排成一行
这是最基础的横向排列

方式

把容器设成 flex，子元素就会沿着主轴排成一行。这个例子里只看“横着排”，别先管别的细节。

这是控制横排。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  max-width: 560px;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
}
Preview
03 / 一列排列

把元素排成一列
就是从上到下堆起来

方式

把主轴改成纵向，元素就会从上到下排。这是很多表单、步骤条、详情区最常见的排列方式。

这是控制一列。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: flex;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
}
Preview
04 / 3x3 卡片

先把卡片排成 3 列
每列再堆 3 张

这里可以把它理解成二维数组：外层是 3 个列，列里面各自再放 3 张卡片。这个写法的前提是，你已经预先知道有几列、每列放哪些卡片。

const cards = [
  [card1, card2, card3],
  [card4, card5, card6],
  [card7, card8, card9],
]

这是3x3 卡片。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card-list {
  display: flex;
  gap: 12px;
}
.card-column {
  display: flex;
  flex-direction: column;
  gap: 12px;
  flex: 1;
}
.card {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff7e8;
  box-sizing: border-box;
  min-height: 72px;
  display: flex;
  align-items: center;
  justify-content: center;
  width: 100%;
}
Preview
05 / 先三列，再一行

还是用 flex
先三列，再让最后一块独占一行

方式

这里继续只用 flex 和 block。前面三块并排，最后一块改成独占整行，就能做出常见的“三列后接说明块”。但这种排法也还是建立在“你提前知道有哪几块”的前提上。

这是三列后再占满一行。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: flex;
  flex-wrap: wrap;
  gap: 12px;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  flex: 0 0 calc((100% - 24px) / 3);
}
Preview
06 / 引入 grid

卡片一旦变成动态的
就该让 grid 接手了

方式

前面的列式写法更像是你先把卡片分好组，再一列一列渲染出来。但真实项目里，卡片数量经常是动态的，接口回来几张并不确定。这时候继续手动分成二维数组就很别扭，grid 更适合接管这种“动态卡片自动排版”的场景。

这是引入 grid 的动态卡片。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: grid;
  gap: 12px;
  grid-template-columns: repeat(3, minmax(0, 1fr));
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff7e8;
  min-height: 72px;
  display: flex;
  align-items: center;
  justify-content: center;
}
Preview
07 / 固定行，列不定

行数可以先定死
列数交给内容自己往右长

方式

grid 也能反过来用：你先固定行数，比如固定 2 行；至于最后会长出几列，不用手算，卡片数量到了就会自动往右补列。这里你可以直接改 rows、columns 和 auto-flow，自己切换“固定行”和“固定列”。

这是固定 2 行，列数自动增长。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: grid;
  gap: 12px;
  grid-template-rows: repeat(2, minmax(0, 1fr));
  grid-auto-flow: column;
  grid-auto-columns: minmax(140px, 1fr);
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff7e8;
  min-height: 72px;
  display: flex;
  align-items: center;
  justify-content: center;
}
Preview
08 / 固定行，列自动变

把效果单独看一遍
固定 2 行，列自动往右长

结果

这一节不让你改参数，只看最终结果。关键点是：你先定死 2 行，后面再加卡片时，`grid` 会继续往右长出新列，而不是往下堆新行。

这是固定 2 行的最终效果。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: grid;
  gap: 12px;
  grid-template-rows: repeat(2, minmax(0, 1fr));
  grid-auto-flow: column;
  grid-auto-columns: minmax(140px, 1fr);
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff7e8;
  min-height: 72px;
  display: flex;
  align-items: center;
  justify-content: center;
}
Preview
