布局控制
东西怎么排

这一章只看“位置关系”。横排、竖排、空隙、对齐、分列，这些都不靠手挪，而是靠布局规则。把这层直觉建立起来，页面就不会再乱堆。

本页讲什么元素怎么排列

01 / 核心

先记住一句话
布局决定元素彼此的位置关系

核心

字号、颜色、边框，是在管“长什么样”；布局是在管“彼此怎么站”。横着排还是竖着排，中间留多大空，一行分成几列，都是布局属性在决定。

02 / 练习

直接动手改五种布局
横排、空隙、分布、对齐、分列

方式

下面几段都只改一条布局相关的 CSS。先不要急着背定义，先让眼睛记住：属性一变，位置关系就变。

这是控制横排。

display: flex 会让子元素进入同一条主轴。把容器从默认文档流改成 flex，模块就会从上下堆叠切到一排。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  max-width: 560px;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
}
Preview

这是控制间距。

gap 控制子元素之间留多少空。值变大，模块之间就会更松；值变小，它们就会贴得更近。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: flex;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
}
Preview

这是控制分布。

justify-content 控制主轴上的剩余空间怎么分。改成 space-between、center、flex-end，整排模块的站位都会变。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: flex;
  gap: 10px;
  min-height: 88px;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
}
Preview

这是控制对齐。

align-items 控制交叉轴上的对齐。顶部对齐、居中对齐、底部对齐，通常一改就能看出来模块有没有站齐。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: flex;
  gap: 10px;
  min-height: 160px;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
}
.layout-box.tall {
  padding-bottom: 48px;
}
Preview

这是控制分列。

grid-template-columns 控制一行分成几列。把它从一列改成两列、三列，你就会看到卡片怎么自动换成网格。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.layout-demo {
  display: grid;
  gap: 12px;
}
.layout-box {
  padding: 18px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
}
Preview
