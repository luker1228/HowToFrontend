CSS 控制页面长什么样。

如果 HTML 是结构化响应，CSS 就像展示层配置。它决定字号、颜色、间距、边框和 hover 状态，决定“浏览器看起来像不像产品”。

分类：CSS 难度：Beginner 预计阅读：14 min 产物：静态网页

这章先只看“样子”

这一章先不讲布局，只先建立一个基础认识：CSS 主要是在控制页面看起来怎么样。字大一点还是小一点，颜色深一点还是浅一点，卡片是挤还是松，按钮有没有反馈，这些都归 CSS 管。

动手 1：控制字号

改 .card-title 的 font-size。比如把 16px 改到 55px，看看标题会不会更显眼。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0; line-height: 1.6; }
Preview

动手 2：控制留白

改 .card 的 padding，从 8px 往上调。这个值越大，卡片里的内容看起来越松。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0; line-height: 1.6; }
Preview

动手 3：控制方向

把 .card-actions 的 flex-direction 从 row 改成 column，看按钮从横排变成竖排。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-button {
  padding: 8px 14px;
  border: 1.5px solid #1a1a1a;
  border-radius: 8px;
  background: #fff;
  cursor: pointer;
}
Preview

动手 4：控制 hover

在 CSS 最后加一条 .card-button:hover { ... }，改背景或颜色。把鼠标移到按钮上——状态反馈就是这点东西，缺了页面就“死”。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-actions { display: flex; gap: 10px; }
.card-button {
  padding: 8px 14px;
  border: 1.5px solid #1a1a1a;
  border-radius: 8px;
  background: #fff;
  cursor: pointer;
}
Preview

动手 5：控制颜色

这里改的是“编辑”和“删除”两个按钮的颜色，也就是 .card-button 这一块。可以直接试这 5 个色值：#1a1a1a、#4b5563、#0f766e、#b45309、#b91c1c。

CSS 
body {
  font-family: system-ui, sans-serif;
  margin: 24px;
  background: #f5f4ee;
  color: #1a1a1a;
}
.card {
  padding: 16px;
  border: 1.5px solid #1a1a1a;
  border-radius: 12px;
  background: #fff;
  max-width: 360px;
}
.card-title { font-size: 22px; margin: 0 0 8px; }
.card-meta { margin: 0 0 12px; color: #6b6b6b; }
.card-body { margin: 0 0 16px; line-height: 1.6; }
.card-actions { display: flex; gap: 10px; }
