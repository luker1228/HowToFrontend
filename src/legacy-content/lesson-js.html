JavaScript 让页面动起来。

这一章不讲大段语法，只让你直接碰 4 个最常见的交互：点击、输入、添加、删除。左边改 JS，右边立刻看结果。

分类：JavaScript 难度：Beginner 预计阅读：12 min 产物：交互小面板

这章讲什么

JavaScript 的本质很简单：用户做了一个动作，页面收到事件，然后你改一点状态或内容，界面马上跟着变。先把这个过程看懂，再学框架会顺很多。
