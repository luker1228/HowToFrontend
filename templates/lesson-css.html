CSS 不是调颜色，是控制页面长相。

如果 HTML 是结构化响应，CSS 就像展示层配置。它决定字号、间距、颜色、边框和 hover 状态——先用这些把”样子”做对，布局系统后面再讲。

分类：CSS 难度：Beginner 预计阅读：14 min 产物：静态网页

先建立”样式旋钮”思维

后端同学常把 CSS 看成杂乱的属性列表。更有效的方法是把它看成一组旋钮：文字大小、颜色深浅、边框粗细、间距松紧、hover 反馈。这章先只看”样子”，布局系统后面再碰。

CSS 可以控制哪些样式

先只看”样子”——颜色、字号、间距、边框、圆角、阴影。布局系统（Flex / Grid / 响应式）后面再讲。

  • color / background：文字颜色和区块背景
  • font-size / font-weight / line-height：文字大小、粗细和行高
  • border / border-radius：边框和圆角
  • margin / padding：外部距离与内部留白
  • box-shadow：阴影，让卡片浮起来
  • hover / transition：鼠标移上去的变化，最轻量的交互反馈

动手试：拧这些旋钮，看页面长相怎么变

每张卡片只放开一个旋钮——被高亮的属性行可以直接改，其余样式锁定只读，让你专注在「这一个属性到底改了什么」。右边是只属于这一张卡片的实时预览。

要点：每个属性就是一个旋钮。你改的值，决定“浏览器看起来像不像产品”。
