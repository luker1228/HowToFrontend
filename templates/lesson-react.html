见多识广：先认组件，再会写提示词。

这章不讲框架，也不讲实现细节。目标只有一个：让后端程序员看到一个界面块时，知道它叫什么、常用来干嘛、写提示词时该怎么叫它。

分类：组件认知 难度：Beginner 预计阅读：10 min 产物：组件词汇表

这章讲什么

很多后端同学不是不会提需求，而是不知道前端那个东西叫什么。你说“右边滑出来那层”，前端叫 Drawer；你说“中间弹出来确认那层”，前端叫 Modal；你说“上面能切换几个页签”，前端叫 Tabs。把这些词补上，提示词质量会立刻提高。

组件长相

这一页的读法很简单：先看长相，再记名字，最后记住它一般在什么场景下出现。你以后写提示词，直接用这些词。

Navigation
Acme Admin
控制台用户订单
页面顶部放一个 Navbar，左侧是 Logo，中间是主导航，右侧是搜索或登录入口。

Navbar

这是啥？ 顶部导航栏。通常横着放在页面最上面。

前端术语： 顶部导航、主导航、品牌区、右侧操作区。

常用来干嘛： 放 Logo、一级菜单、搜索、登录入口、头像菜单。

提示词写法： 页面顶部放一个 Navbar，左侧是 Logo，中间是导航，右侧是搜索和用户头像。
Navigation

用户管理
订单管理
系统设置
页面左侧放一个 Sidebar，包含分组标题、当前高亮菜单和折叠态。

Sidebar

这是啥？ 侧边栏。通常竖着放在页面左边，用来切换主功能区。

前端术语： 侧边导航、菜单组、当前高亮项、折叠态。

常用来干嘛： 管理后台导航、工作台入口、系统菜单切换。

提示词写法： 左侧放一个 Sidebar，支持当前菜单高亮，适合后台系统。
Action

点击一个按钮试试。

右上角放一个主按钮“新建用户”，旁边放次按钮“导出”，危险按钮用来删除。

Button

这是啥？ 按钮。页面里最常见的动作触发器。

前端术语： 主按钮、次按钮、危险按钮。

常用来干嘛： 提交表单、保存修改、删除数据、打开弹窗。

提示词写法： 右上角放一个主按钮“新建用户”，旁边放次按钮“导出”。
Input

当前输入：zhangsan@example.com

顶部放一个搜索输入框，placeholder 是“搜索用户名 / 手机号”，输入时实时过滤列表。

Input

这是啥？ 输入框。用户往里面填文字。

前端术语： 输入框、占位文案、错误态、禁用态。

常用来干嘛： 搜索、录入名称、输入邮箱、填写参数。

提示词写法： 顶部放一个搜索输入框，placeholder 是“搜索用户名 / 手机号”。
Select
表格上方放一个角色 Select，可选“全部 / 管理员 / 普通用户 / 访客”。

Select

这是啥？ 选择器。点一下展开选项，再选一个值。

前端术语： 下拉选择器、单选、多选、默认项。

常用来干嘛： 筛选角色、切换状态、选时间范围、选部门。

提示词写法： 表格上方放一个角色 Select，可选“全部 / 管理员 / 普通用户”。
Navigation

当前面板：全部

列表上方加 Tabs，支持“全部 / 进行中 / 已完成”切换。

Tabs

这是啥？ 标签页。几块内容之间的平级切换。

前端术语： 标签页、当前激活项、切换面板。

常用来干嘛： 切换分类、切换状态、切换不同内容视图。

提示词写法： 列表上方加 Tabs，支持“全部 / 进行中 / 已完成”切换。
Data
姓名角色状态
张三管理员启用
李四访客禁用
主内容区放一个用户 Table，列有姓名、角色、状态、创建时间、操作。

Table

这是啥？ 表格。后台系统里最常见的数据展示方式。

前端术语： 列、行、操作列、表头、分页、筛选。

常用来干嘛： 用户列表、订单列表、任务列表、日志列表。

提示词写法： 主内容区放一个用户 Table，列有姓名、角色、状态、创建时间、操作。
Feedback
确认删除？

删除后不可恢复。

点击删除按钮后弹出确认 Modal，包含标题、说明、取消和确认按钮。

Modal

这是啥？ 弹窗。内容浮在页面中间，通常要先处理它。

前端术语： 弹窗、确认框、遮罩层、关闭按钮。

常用来干嘛： 删除确认、二次确认、关键设置、补充说明。

提示词写法： 点击删除按钮后弹出确认 Modal，包含标题、说明、取消和确认按钮。
Overlay
用户详情

手机号、角色、最后登录时间

点击表格行后，从右侧打开一个 Drawer 展示用户详情。

Drawer

这是啥？ 抽屉。从侧边滑出来的一层，常见于右侧。

前端术语： 右侧抽屉、侧滑层、详情面板。

常用来干嘛： 看详情、改资料、补表单，不想跳页面时很常见。

提示词写法： 点击表格行后，从右侧打开一个 Drawer 展示用户详情。
Paging

当前页：1

表格底部加 Pagination，显示当前页、总条数和上一页下一页。

Pagination

这是啥？ 分页器。数据太多时，用它切到下一页。

前端术语： 分页器、当前页、上一页、下一页、总条数。

常用来干嘛： 表格翻页、文章列表翻页、搜索结果翻页。

提示词写法： 表格底部加 Pagination，显示当前页、总条数和上一页下一页。

提示词怎么写

错误写法：

右边出来一个东西，中间弹一个框，上面再放几个切换按钮。

更好的写法：

右侧打开一个 Drawer 展示详情；删除时弹出 Modal 确认；列表上方放 Tabs 切换状态；主内容区用 Table 展示数据，底部带 Pagination。
